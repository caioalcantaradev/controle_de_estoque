## 🚀 Como Executar Localmente

### Pré-requisitos
- Node.js 16+ 
- npm ou yarn

### Instalação
```bash
# Clone o repositório
git clone https://github.com/caioalcantaradev/controle_de_estoque.git
cd controle_de_estoque

# Instale as dependências
cd frontend
npm install

# Execute o projeto
npm start
```
